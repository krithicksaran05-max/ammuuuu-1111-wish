# Luxury 11:11 Digital Experience — For Ammuuu ❤️

A premium, cinematic digital love letter experience built with Next.js, React 19, Tailwind CSS v4, Framer Motion, and HTML5 Canvas.

## Features

- ✨ **Cinematic Cosmic Loader**: Beautiful galaxy particle field with sequential text reveals leading to a glowing "11:11" clock and liquid glass "Open My Wish" button.
- 🌌 **Living Night Sky (60fps)**: Starfield Canvas rendering 3D parallax stars, diagonal shooting stars, undulating aurora borealis, fireflies, rising blue hearts, and falling petals.
- 🖼️ **Interactive Glass Photo Frame**: Floating frame with dynamic border glow, liquid reflection, scroll zoom, and parallax mouse tracking.
- 📜 **Scroll-Revealed Love Letter**: Elegant Cormorant Garamond / Playfair Display typesetting with custom text shimmer animations fading in sentence-by-sentence.
- ⏰ **11:11 Magic Event System**: Automatically triggers music, heart/petal confetti bursts, intensified auroras, and a beautiful overlay when the clock strikes 11:11 AM/PM.
- 🎨 **Desktop Cursor Sparkles**: Multi-point stars trailing the cursor on desktop and touch ripple effects on mobile.
- 🥚 **Hidden Surprises**: Tap 11 times anywhere to trigger a starburst shower, or hold press anywhere to reveal a secret note.

## Local Development

To run the application locally:

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- **Photo**: Swap your custom photo at `public/images/my-photo.png`.
- **Music**: Swap your custom song at `public/music/my-song.mp3`.

## Deployment

Deploy directly to Vercel:

```bash
# Install vercel CLI
npm install -g vercel

# Deploy
vercel
```
